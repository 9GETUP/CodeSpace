- 如何向开源项目提交代码
    - fork 大佬的项目
        就来到自己的仓库，有提交权限
    - 发现了大佬的bug 或他要我们做的job
        把活干完，完成测试
        add commit push
    - 提交到我们自己的远程仓库
    - 点进这次提交 create pull request
        描述问题 怎么解决的 截图 提交
    - 大佬会收到消息，检查 没有问题，merge
